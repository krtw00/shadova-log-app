案A: 折りたたみサイドバー型

  • サイドバーを折りたたんで画面を広く使える
  • 対戦記録がダッシュボード形式で一目で状況把握
  • 統計カードで今日の戦績を常時確認
  • 現在の実装に近く、移行がスムーズ
レイアウト構成
┌────┬──────────────────┐
│    │     Header       │
│Side├──────────────────┤
│bar │                  │
│    │   Main Content   │
│    │   (Dashboard)    │
└────┴──────────────────┘

案B: トップナビ + タブ型

  • 上部固定ナビで縦方向を最大活用
  • 対戦記録内でさらにタブ分け（ダッシュボード/履歴/分析）
  • ヒーロースタッツで今日の成績を大きく表示
  • サイドバーがないため全幅使用可能

├─────────────────────────┤
│                         │

│                         │

案C: 3カラム統合ダッシュボード型

  • 左: アイコンのみのミニナビ（省スペース）
  • 中央: 対戦記録リスト（メインコンテンツ）
  • 右: 統計/デッキパネル（常時表示）
  • 対戦記録と統計を同時に確認できるダッシュボード運用に最適
